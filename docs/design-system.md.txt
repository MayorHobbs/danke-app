## 1. DESIGN PHILOSOPHY

**Core Principles:**
- **Clarity over decoration** - Information-first, no visual noise
- **Intentional minimalism** - Every pixel serves a purpose
- **Philosophical weight** - Design conveys depth and thoughtfulness
- **Calm technology** - Widget doesn't scream for attention

**Inspirations:**
- Dan Koe's website aesthetic (clean, typography-focused)
- Brutalist web design (function over form)
- Apple Human Interface Guidelines (system integration)
- Dieter Rams principles (less but better)

**Serif (Quotes):**
- **iOS:** Georgia (system serif, philosophical weight)

## 8. ACCESSIBILITY

### 8.1 Requirements

**Text Contrast:**
- All text must meet WCAG AA (4.5:1 for body, 3:1 for large)
- Quote (#E8DCC4 on #1A1A1A): **8.2:1 âœ“**
- Action (#F5F5F5 on #1A1A1A): **12.6:1 âœ“**

**Dynamic Type:**
- Support iOS Dynamic Type (automatically scales)
- Test at largest accessibility sizes

**VoiceOver / TalkBack:**
```
Widget Accessibility Label:
"Quote: [quote text]. Action: [action text]. 
 From [source]. Estimated time: [duration]."

Checkbox:
"Mark action as complete. Currently unchecked."
```

**Reduced Motion:**
- Respect system setting
- Replace animations with instant transitions

### 8.2 Color Blind Considerations

**Domain Tags:**
- Don't rely solely on color
- Include text labels always
- Test with:
  - Protanopia (red-blind)
  - Deuteranopia (green-blind)
  - Tritanopia (blue-blind)

**Gold (#C9A961) vs Green (#7FA66A):**
- Sufficient luminance difference
- Both readable against dark background

## 9. RESPONSIVE DESIGN

### 9.1 Widget Size Variants

**iOS:**
```
Small (2x2):  Primary design (ship first)

**Quote Text:**
```
Max lines: 4

Prioritize: First 3 lines + "..." over squishing font
```

**Action Text:**
```
Max lines: 3
Overflow: Truncate with "..."
Never: Reduce font size below 13pt
```

**Metadata:**
```

## 12. BRAND VOICE (UI Copy)

**Principles:**
- **Direct, not clever:** "Mark Complete" not "Got it!"
- **Respectful, not casual:** "Reflect on today" not "How'd it go?"
- **Philosophical, not motivational:** Avoid "You can do this!"

**Examples:**

**Good:**
- "What did you learn from today's action?"
- "Choose your area of focus"
- "Today's wisdom"

**Bad:**
- "You crushed it! ğŸ‰"
- "Level up your mindset!"
- "Daily dose of inspiration"
